# プライバシーポリシー

れんらくボード (contact-board) は、よく連絡する家族の名前と短いメモを大きな文字で表示する Chrome 拡張です。

## 収集・保存するデータ

- ユーザーが入力した名前
- ユーザーが入力した短いメモ
- 7日トライアル判定に使う初回起動日時
- 将来の Premium 判定に使うローカル状態

## データの保存場所

データは Chrome の `storage` API を使い、利用中のブラウザ内に保存されます。

## 外部送信について

本拡張は API 通信やネットワーク通信を行いません。入力されたデータを外部サーバー、第三者サービス、開発者に送信しません。

## 権限

本拡張が使用する Chrome 権限は `storage` のみです。閲覧中の Web サイト内容へアクセスする権限や、外部ホストへのアクセス権限は使用しません。
